# 数据报表

## 功能介绍
数据洞察：支持跨平台，多维度，全方位的数据报表查询

<img src="./adreport.png" alt="数据报表.png" style="zoom:50%;"/>

 * 目前已经支持广告平台：腾讯广告-AMS，今日头条-巨量引擎
 * 广告主/广告计划/广告组/创意数据：包括花费、点击量、点击单价等
 * 素材报表数据：包括视频，图片维度的展示数、点击数、点击单价等
 * 多平台的报表数据聚合对比
